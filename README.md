Descripción General
Este proyecto marca la culminación de la primera etapa del curso de fundamentos de programación. Es una aplicación web interactiva que tiene como objetivo consolidar los conocimientos esenciales de HTML, CSS, Bootstrap y JavaScript. La aplicación simula una galería de imágenes completa, desde una vista de inicio y galería con filtros, hasta una página de detalle de imagen y una sección "Sobre nosotros" con un formulario de contacto validado.

Tecnologías Clave Utilizadas
HTML5: Estructura semántica con etiquetas como <header>, <main>, <section> y <footer>.

CSS3 y Bootstrap: Diseño responsivo y coherente, aplicación de estilos de tipografía, paleta de colores, y maquetación de la interfaz de usuario.

JavaScript: Lógica de programación para añadir dinamismo, incluyendo la funcionalidad de filtrado de imágenes por categoría, la vista detallada de cada elemento y la validación del formulario de contacto.

GIT/GitHub: Control de versiones, organización y gestión del proyecto (ideal para práctica de trabajo en equipo).

Características Principales
Diseño Responsivo: Implementado con Bootstrap para una correcta visualización en diversos dispositivos.

Galería Funcional: Muestra miniaturas de imágenes y permite filtrarlas por categorías.

Vistas Dinámicas: Incluye páginas de Inicio, Galería, Detalle de Imagen y Sobre Nosotros.

Interactividad: Funciones en JavaScript para manejo de datos (filtros) y feedback al usuario (validación de formulario).

Objetivos de Aprendizaje Reforzados
Este desafío ha permitido aplicar la lógica de programación, desarrollar habilidades de resolución de problemas y practicar la organización de tareas en un entorno de desarrollo real. Es una demostración de la capacidad de construir una interfaz de usuario estructurada, estilizada y funcional.
